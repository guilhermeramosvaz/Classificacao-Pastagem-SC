Mapeamento da Pastagem no estado de Santa Catarina


Figura 1: Lapig(11/2025).

Introdução

Trabalho de mapeamento das pastagens do estado de Santa Catarina para o ano de 2024 realizado pela parceria entre Lapig e Remapgeo. Utilizando imagens do satélite Sentinel 2A/B com 10 metros de resolução espacial, dados de referência do Mapbiomas e Global Pasture Watch, foi possível gerar uma classificação supervisionada para todo o estado, realizada com o Algorítmo Random Forest, descrita em Visão Geral.

Bases de dados

Foram utilizados como dados de referência os dados de pastagem Mapbiomas e Global Pasture Watch do Land & Carbon Lab para gerar os pontos utilizados.

Requisitos

  • Google Earth Engine;
  • Python v3.12 ou superior;
  • Numpy v2.2.4 python package;
  • Gdal binários v3.10.3;
  • R v4.5.2;
  • Qgis v3.22 ou superior;
  • ThRasE v26.x ou superior;
